# 🎯 Jogo de Adivinhação em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

---

## 🚀 Funcionalidades

- Geração de número aleatório entre 0 e 10
- Sistema de tentativas limitadas
- Validação de entrada do usuário
- Menu interativo (iniciar ou sair)
- Mensagens de acerto e erro

---

## 🎮 Como funciona

O programa gera um número aleatório e o jogador precisa adivinhar qual é.

- Você tem **5 tentativas**
- Se acertar → vitória 🎉
- Se errar todas → o jogo revela o número

---

## 🧠 Tecnologias utilizadas

- Python
- Biblioteca `random`
- Biblioteca `os`
